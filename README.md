
# MovingSigmas

A GIF searching app made with giphy API




## Installation

clone repo 

```bash
  cd movingsigmas
  expo Install
  expo run android
```
    
## Features

- Portait/landscape
- search
- lazy loading/infinite scroll
- redux
- navigation v6
